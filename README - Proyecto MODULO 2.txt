README - Proyecto MODULO 2

Título: "Interior Design Showcase"

Descripción:
Este proyecto es una página web responsive desarrollada con HTML5 y CSS3, que presenta una introducción a una marca de diseño de interiores y una sección de productos destacados que se despliega mediante un checkbox oculto.

Estructura de carpetas:
├── index.html
├── css/
│   ├── reset.css
│   ├── variables.css
│   ├── fonts.css
│   ├── responsive-main.css
│   └── desktop.css
└── recursos mod-2/
    └── [imágenes del proyecto]

Tecnologías utilizadas:
- HTML5
- CSS3 (Flexbox y Grid)
- Media Queries para diseño responsive

Características principales:
- Diseño responsive para escritorio y móvil
- Sistema de tarjetas de productos con grid adaptativo
- Galería de imágenes tipo mosaico en layout de escritorio
- Botón tipo desplegable para mostrar los productos
- Uso de variables CSS y reset.css para mejor consistencia
- Fuentes personalizadas importadas desde fonts.css

Modo de visualización recomendado:
- Abrir `index.html` en navegador de escritorio y móvil
- También se puede usar DevTools (F12) para verificar responsividad
